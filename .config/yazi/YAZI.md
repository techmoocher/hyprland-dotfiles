# YAZI - A blazing fast TUI file manager written in Rust

*Refer to [this repo](https://github.com/sxyazi/yazi.git/) for more*

## Installation

### Arch

```bash
sudo pacman -Syu --needed yazi file ffmpeg 7zip jq poppler fd ripgrep fzf zoxide resvg wl-clipboard imagemagick ttf-jetbrains-mono-nerd ttf-firacode-nerd
```

### Debian-based distros

```bash
sudo apt update
sudo apt upgrade
sudo apt install yazi ffmpeg 7zip jq poppler-utils fd-find ripgrep fzf zoxide imagemagick
```
